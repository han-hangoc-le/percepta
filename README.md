# CV Script for only data box overlay

**Usage:**

- This code is for testing the CV logic on laptop camera (segmentation + rough info overlay + toggleable contour mode).
- Please ensure the model yolov8n-seg.pt is in the same directory as this script. If not, the model can be downloaded on website.
- Download neccessary dependencies
- `python testcv.py` in the terminal to test run the script on laptop
- Press 'o' to toggle contour mode, and press 'q' to quit

**Why it's here:**

This script is here for further implementation of mode "biologist", "ecologist", and so on - those that currently has trouble with GenAI overlay.
